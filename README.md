
#  Cookie Cats: A/B Testing and Causal Inference Analysis

Welcome to our statistical analysis project! I analyzed the results of A/B test for the popular mobile puzzle game *Cookie Cats*.

##  Project Objective
The main objective of this project is to analyze the impact of moving the first "door" in the game from level 30 to level 40. I investigated how this change affected player retention rates (1-day and 7-day retention rates) using data-driven statistical methods.

##  Basic Methodologies
Instead of just looking at basic averages, I applied advanced analytical techniques to provide reliable business decisions:
* *A/B Testing:* I formulated null and alternative hypotheses to measure the actual difference between the control (door_30) and test (door_40) groups.

* *Causal Inference:* The actual causal effect of door placement on user behavior was evaluated.
* *Statistical Significance:* Appropriate statistcal tests (e.g., Chi-Square, Permutation tests) were used to verify the reliability of retention metrics.

##  Tools and Technology Stack
* *Python* * *Pandas and NumPy:* For data manipulation and exploratory data analysis (EDA).

* *Statistical Analysis:* Statistical logic was applied to accurately interpret retention and engagement metrics.
